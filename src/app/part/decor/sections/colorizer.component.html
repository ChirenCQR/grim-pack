The majority of Grim Decor is blocks that can be colored to look like many other blocks. All of these are colorizer blocks and require some base materials to make first. All colorizer blocks work the same with dying and clearing. You can right click with a block in hand to paint it and to clear either break it or use a brush. There are two modes in the config. 'Colorizers Consume Block' this option allows you to change if colorizers consume blocks when painted, true for this false for free paints.

Hardened Wood

This is used in the construction of colorizer blocks. This acts as a harder wood that can then be influenced easily with dyes.


Colorizer

The colorizer allows it to take on the texture of many different types of blocks. Once crafted simply right click on it with a compatible block and it should change right away. To clear the texture you can break it and replace it or craft a brush.


Colorizer Light

This has the same properties of a colorizer except that it emits light.


Brush

The brush is used to clear a colorizer. This works for any colorizeable block and in both modes. So, where it consumes an item and where it doesn't.
