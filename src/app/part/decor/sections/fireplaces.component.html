If you are a fan of burning things or really like different fireplaces then this has what you want. Decor contains many different types of fireplaces. All fireplaces are able to be burnt and some fire will appear when you use flint and steel on them. This fire can be put out with a punch.

Burning Wood

This is a special type of wood imbued with Netherrack that allows for a nice look but also allows it to burn indefinitely. This is used in most of the recipes for Fireplaces.


Fireplaces

Fireplaces can connect to each other and will also make chimneys will placed on them emit smoke.


Chimneys

Chimneys will emit smoke if above a fireplace or stove. These can be stacked together to create large smoke stacks. Chimneys will only emit smoke if the fireplace is active and burning.


Firepits

Firepits are another fireplace that looks quite nice. There is a config option to disable the net around the firepit.


Stove

Stoves are closed off fireplaces. These look quite nice and have a nice protective cage so you don't burn yourself. These also can make smoke emit from chimneys if they are burning.


Firerings

If you like to go camping then this can work a portable campfire.


Grills

Grills are a way to cook food over a fire. These can be right clicked to open a interface to add your ingredients to cook and fuel to burn. The grill is very configurable, it has options for infinite grill fuel and possible recipes can be changed via the config file as well. Depending on if you have consume blocks on or off the speed might change. If you have it true then depending on the material you use for the grill it will change the speed. If it is turned off then a flat rate for all blocks will be applied.
