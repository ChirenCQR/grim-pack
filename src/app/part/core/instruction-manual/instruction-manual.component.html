Instruction Manual

The only thing that Core includes new is the Instruction Manual. The instruction manual contains all of the information here but directly in Minecraft so that it is easy to access and all the information will be accurrate and up to date since it has access to all current recipes and build instructions. The instruction manual allows you to view information on everything in the Grim Pack. This includes blocks, items, entites, world gen and more.

Crafting:

How to use:

  • Right click to open.
  • Use the arrows at the bottom for navigating the pages.
  • Click on the mods name to go to the sub-section for that mod.
  • It will save the last page you were on while ingame.
  • When in a crafting or furnace recipe you can pause the recipe using the pause button.
  • You can only pause if it is a recipe that moves.
